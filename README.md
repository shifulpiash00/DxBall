# DX-Ball (OpenGL C++ Project)

A classic **DX-Ball / Brick Breaker** style game implemented in **C++ using OpenGL (GLUT)**. The project includes animated menus, sound effects, background music, and multiple visual assets. Built and tested on **Windows** with **Visual Studio**.
